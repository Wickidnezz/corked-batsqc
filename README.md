# Corked Bats Website

Static starter site for Corked Bats.

## Files

- `index.html` — homepage
- `styles.css` — design and layout
- `script.js` — mobile menu and footer year
- `thank-you.html` — Netlify form success page
- `assets/logo.png` — logo

## Netlify settings

If these files are at the root of your GitHub repository:

- Build command: leave blank
- Publish directory: `.`

If these files are inside a folder, publish that folder instead.
